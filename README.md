# Tenta Tsuji Homepage

Static files for `https://ten10call.github.io/`.

## Files

- `index.html`: homepage content
- `styles.css`: visual design
- `assets/hero-cosmology.svg`: hero background image

## Publish

Create a GitHub repository named `ten10call.github.io`, put these files on the default branch, and enable GitHub Pages if it is not enabled automatically.
